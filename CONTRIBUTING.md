# Contributing

Thank you for your interest in contributing! All contributions are welcome.

## How to Contribute

### Reporting Bugs

1. Check if the bug has not already been reported in [Issues](https://github.com/mytoandeptrai/porforlio.git/issues)
2. Open a new issue using the **Bug Report** template
3. Include as many details as possible: steps to reproduce, expected vs. actual behavior, screenshots

### Suggesting Features

1. Open an issue using the **Feature Request** template
2. Clearly describe the improvement and the problem it solves

### Submitting Pull Requests

1. Fork the repository
2. Create a branch from `master`: `git checkout -b feat/my-feature`
3. Make your changes following the project's standards
4. Commit with semantic messages in English (e.g., `feat: add animation to hero`)
5. Push to your fork: `git push origin feat/my-feature`
6. Open a Pull Request describing your changes

## Project Standards

- **Semantic commits** in English (`feat:`, `fix:`, `refactor:`, `docs:`, `chore:`)
- **TypeScript** with strict typing
- **React components** following SOLID principles and proper componentization
- **Tailwind CSS** for styling
- Run `npm run lint` before submitting your PR

## Development Environment

```bash
git clone https://github.com/mytoandeptrai/porforlio.git
cd myPortfolio
npm install
npm run dev
```
